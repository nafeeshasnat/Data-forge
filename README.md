# DataForge

DataForge is a powerful and intuitive application for generating synthetic student datasets. It provides a flexible platform for creating realistic and customizable data for educational research, software testing, and data analysis purposes.

- **Data Export:** Export the generated dataset in JSON format for easy integration with other tools and analysis pipelines.
